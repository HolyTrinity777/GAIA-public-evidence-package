# PRIVATE_ACCESS

This repository contains a **public evidence package** for GAIA — Global Governance Swarm.

This document defines what is intentionally excluded from the public release and what may be discussed only under NDA or separate written agreement.

## Excluded from Public Release

The following materials are private and are not included in the public evidence package:

- Full agent implementations
- Internal architecture
- Weights and scoring logic
- Internal metrics
- Optimization strategies
- Recovery mechanisms
- Arbitration logic
- Deployment infrastructure
- Topology details
- Proprietary runtime internals
- Any unredacted trace, log, example, or artifact that reveals private mechanics

## NDA Boundary

Access to excluded materials is available only under NDA or separate written agreement.

If a reviewer needs to evaluate the private core, the discussion must stay bounded to the NDA scope. The public repository itself should not contain implementation details that would expose the edge of the system.

## Public / Private Rule

If a file, screenshot, or example reveals:

- how the system achieves its behavior,
- how agents are coordinated internally,
- how hidden decisions are made,
- or how the runtime is protected from manipulation,

then it belongs in the private boundary, not the public package.

## Intended Use of the Public Package

The public package is designed to make the system's observable behavior reviewable without exposing proprietary internals. It is evidence of operation, not source disclosure.
